🚗 Car Price Prediction using Machine Learning

📌 Overview

This project predicts the selling price of cars using machine learning techniques. 
It analyzes various factors such as year, fuel type, and present price to estimate car value.

🎯 Objectives

Build a machine learning model to predict car prices
Compare multiple models for performance
Identify key factors influencing car price

🛠️ Tools & Technologies

Python

Pandas, NumPy

Matplotlib, Seaborn

Scikit-learn

📂 Dataset

The dataset contains information about cars such as:

Year of manufacture

Present price

Fuel type

Transmission

Selling price


🤖 Models Used

Linear Regression (R²: 0.99)

Random Forest (R²: 0.96)

🔍 Key Insights

The most important feature influencing price is Present_Price

High accuracy is due to strong correlation between features

Model performance drops when key feature is removed


🚀 How to Run

Clone the repository

Install dependencies

Run the notebook

📁 Project Structure

Data/ → dataset

Notebook/ → model building

Graphs/ → visualizations

Model/ → saved ML model
